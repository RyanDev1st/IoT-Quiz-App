# IoT Quiz App

## Instructions for Use

To run this application on your local machine:

1. **Download the project**: Clone or download this repository to your local machine as a ZIP file and extract it.
2. **Open the folder**: Navigate to the directory where you extracted or cloned the project.
3. **Run the app**: Double-click on the `index.html` file to open it in your web browser.

**Important Note**: This application is designed to be run locally directly via `index.html` and **should not be deployed online**. All data and quiz logic run entirely in your local browser environment.
